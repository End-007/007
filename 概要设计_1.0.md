## 1. 引言

### 1.1 编写目的

本文档旨在详细描述学管平台的总体设计方案，为后续的详细设计和编码实现提供指导。本文档的读者包括系统架构师、开发人员、测试人员等。

### 1.2 项目背景

现在学院里的数据都分散在不同地方，老师和学生想查个东西很不方便，统计数据也很麻烦。本平台旨在解决数据分散、查询不便、统计困难等问题，通过数据大屏直观展示学院教学、科研和学习的整体情况。

### 1.3 术语定义

- **数据大屏**：平台的核心展示界面，用于直观展示各类统计数据
- **外部数据导入**：从外部系统（如教务系统）导入数据到本平台
- **数据统计**：对数据进行汇总、分析和展示
- **数据查询**：根据条件查询数据

## 2. 任务概述

### 2.1 目标

本平台的主要目标是：

1. 实现教学、科研和学习数据的统一管理和可视化展示
2. 提供数据大屏，直观展示学院的整体情况
3. 提供数据统计和查询功能，方便用户深入了解数据
4. 系统架构简单，便于学生实现
5. 操作流程简洁，用户体验良好

### 2.2 用户特点

平台主要面向三类用户：

- **管理员**：负责平台管理、数据导入和用户权限管理
- **教师**：查看自己教学相关的数据统计和学习情况，导入数据
- **学生**：查看自己的课程、成绩、学习进度等个人数据

### 2.3 假定和约束

1. 数据由外部系统导入
2. 系统架构简单，使用成熟的技术栈
3. 代码结构清晰，便于维护和扩展

## 3. 总体设计

### 3.1 系统架构

本系统采用前后端分离的架构，主要包括以下几个部分：

```
┌─────────────────────────────────────────┐
│           前端展示层                      │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │数据大屏│ │统计查询│ │个人中心│          │
│  └──────┘  └──────┘  └──────┘          │
└─────────────────────────────────────────┘
                    ↑↓
┌─────────────────────────────────────────┐
│           后端服务层                      │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │数据服务│ │统计服务│ │用户服务│          │
│  └──────┘  └──────┘  └──────┘          │
└─────────────────────────────────────────┘
                    ↑↓
┌─────────────────────────────────────────┐
│           数据存储层                      │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │ MySQL│  │ Redis│  │文件存储│          │
│  └──────┘  └──────┘  └──────┘          │
└─────────────────────────────────────────┘
```

## 4. 数据库设计

### 4.1 数据库表设计

#### 4.1.1 用户表（user）

| 字段名      | 类型         | 说明                       |
| ----------- | ------------ | -------------------------- |
| id          | BIGINT       | 主键 ID                    |
| username    | VARCHAR(50)  | 用户名，唯一               |
| password    | VARCHAR(100) | 密码（加密）               |
| role        | VARCHAR(20)  | 角色（管理员、教师、学生） |
| status      | VARCHAR(20)  | 状态（正常、禁用）         |
| create_time | DATETIME     | 创建时间                   |
| update_time | DATETIME     | 更新时间                   |

#### 4.1.2 学生表（student）

| 字段名      | 类型        | 说明       |
| ----------- | ----------- | ---------- |
| id          | BIGINT      | 主键 ID    |
| user_id     | BIGINT      | 用户 ID    |
| student_no  | VARCHAR(20) | 学号，唯一 |
| name        | VARCHAR(50) | 姓名       |
| class_name  | VARCHAR(50) | 班级       |
| department  | VARCHAR(50) | 系别       |
| create_time | DATETIME    | 创建时间   |
| update_time | DATETIME    | 更新时间   |

#### 4.1.3 教师表（teacher）

| 字段名      | 类型        | 说明       |
| ----------- | ----------- | ---------- |
| id          | BIGINT      | 主键 ID    |
| user_id     | BIGINT      | 用户 ID    |
| teacher_no  | VARCHAR(20) | 工号，唯一 |
| name        | VARCHAR(50) | 姓名       |
| title       | VARCHAR(50) | 职称       |
| department  | VARCHAR(50) | 所属系     |
| create_time | DATETIME    | 创建时间   |
| update_time | DATETIME    | 更新时间   |

#### 4.1.4 课程表（course）

| 字段名      | 类型         | 说明           |
| ----------- | ------------ | -------------- |
| id          | BIGINT       | 主键 ID        |
| course_no   | VARCHAR(20)  | 课程编号，唯一 |
| course_name | VARCHAR(100) | 课程名称       |
| teacher_id  | BIGINT       | 任课教师 ID    |
| department  | VARCHAR(50)  | 所属系         |
| create_time | DATETIME     | 创建时间       |
| update_time | DATETIME     | 更新时间       |

### 4.2 数据库表关系

- 用户表与学生表：一对一关系
- 用户表与教师表：一对一关系
- 教师表与课程表：一对多关系

## 6. 使用说明

### 6.1 环境搭建

#### 6.1.1 后端环境

1. 安装 JDK 8 或以上版本
2. 安装 Maven 3.6 或以上版本
3. 安装 MySQL 8.0
4. 安装 Redis

#### 6.1.2 前端环境

1. 安装 Node.js 14 或以上版本
2. 安装 Vue CLI

### 6.2 系统部署

#### 6.2.1 后端部署

1. 克隆项目代码
2. 修改配置文件（application.yml），配置数据库和 Redis 连接信息
3. 执行 SQL 脚本，创建数据库和表
4. 使用 Maven 打包项目：`mvn clean package`
5. 运行 jar 包：`java -jar school-platform.jar`

#### 6.2.2 前端部署

1. 克隆项目代码
2. 安装依赖：`npm install`
3. 修改 API 接口地址配置
4. 构建项目：`npm run build`
5. 将构建产物部署到 Nginx 或其他 Web 服务器

### 6.3 功能使用

#### 6.3.1 数据大屏

1. 登录系统后，进入数据大屏页面
2. 可以查看教学、科研、学习和整体概况的统计数据
3. 可以按时间范围和层级筛选数据
4. 可以点击图表查看详细信息
5. 可以导出大屏为图片或 PDF

#### 6.3.2 数据统计与查询

1. 进入数据统计页面，选择统计类型（教学、科研、学习）
2. 设置查询条件，点击查询按钮
3. 查看统计结果，可以导出为 Excel 或 PDF

#### 6.3.3 个人中心

1. 进入个人中心页面
2. 根据用户角色，可以查看相应的数据和功能
3. 可以更新个人信息

#### 6.3.4 数据导入

1. 管理员登录系统后，进入数据导入页面
2. 选择导入类型（学生、教师、课程等）
3. 上传 Excel 文件
4. 系统自动验证数据格式并导入
5. 查看导入结果和错误信息
